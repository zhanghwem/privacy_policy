Privacy Policy

AI Phone Finder is a tool application that helps user quickly find a phone that is behind an object and not easily to be found. The application collects and saves the user's voice samples, and intelligently compares with the voice which the user waken the phone to determine whether the user is looking for the phone and then responds.

This application collects user's voice samples and samples the voice in real time. However, all voice data is only used to waken the phone and will never be used for other purposes. This application does not require any user identity information, and will never reveal user privacy.
