Política de privacidad

Buscador de Telefonos AI es una aplicación de herramienta que ayuda al usuario a encontrar rápidamente un teléfono que está detrás de un objeto y no es fácil de encontrar. La aplicación recopila y guarda las muestras de voz del usuario, y las compara de manera inteligente con la que el usuario despierta el teléfono para determinar si está buscando el teléfono y luego responde.

Esta aplicación recopila muestras de voz del usuario y muestra la voz en tiempo real. Sin embargo, todos los datos de voz solo se utilizan para activar el teléfono y nunca se utilizarán para otros fines. Esta aplicación no se conecta a la red, no requiere ninguna información de identidad del usuario y nunca revelará la privacidad del usuario.
