Política de privacidad

Buscador de Telefonos IA es una aplicación de herramienta que ayuda al usuario a encontrar rápidamente un teléfono que está detrás de un objeto y no es fácil de encontrar. La aplicación recopila y guarda las muestras de voz del usuario, y las compara de manera inteligente con la que el usuario despierta el teléfono para determinar si está buscando el teléfono y luego responde.

Esta aplicación recogerá muestras de voz del usuario y la tomará en tiempo real cuando esté en uso. Sin embargo, todos los datos de voz solo se usan para activar el teléfono para responder las llamadas de los usuarios y nunca se usan para otros fines. Esta aplicación no requiere ninguna información de identidad del usuario y nunca perderá la privacidad del usuario.
