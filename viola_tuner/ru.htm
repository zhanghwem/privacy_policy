Виола Тюнер

Использование телефона для настройки вашей альта. Точный и простой в использовании. Ваш сильный партнер Виола.

Это приложение будет использовать ваш микрофон для записи, а все записанные данные будут использоваться только для частотного анализа и не будут использоваться для других целей. Это приложение не подключено к сети, не требует никакой идентификационной информации пользователя и не раскрывает конфиденциальность пользователя.
